# SQL Practice — Misab Abdul Rashid

## Purpose
A compact, interview-ready collection of SQL exercises and analytics queries showcasing practical capabilities: joins, window functions, CTEs, aggregations, and short case studies.

## Files
- `problems/01-joins.sql` — common join patterns and aggregation examples.
- `problems/02-window-functions.sql` — ranking, rolling sums, lag/lead, percent contribution.
- `problems/03-ctes.sql` — multi-CTE pipelines and recursive CTE example (calendar).
- `problems/04-analytics-case-study.sql` — mini business case: repeat customers, top repeat products, churn signal.
